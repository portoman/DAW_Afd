## for / of
```javascript
for (let tuvariable of uniterable) {
    //código a ejecutar
}
```

## if
```javascript
if ( expresion ) {
    // Código a ejecutar sólo si expresión == true
}
```

## if / else
```javascript
if ( expresion ) {
    // Código a ejecutar sólo si expresión == true
} else {
    // Código a ejecutar sólo si expresión != true
}
```
